Setting up python virtual environmnet
1. Installing virtualenv package
	> pip install virtualenv
2. Open the terminal in the current working directory(i.e. this folder)
	> virtualenv venv
3. Start the virtual environment
	> venv\Scripts\activate
4. Use the requirements.txt file to install all the required packages
	> pip install -r requirements.txt


Running the files
1. Open the terminal in the current working directory(i.e. this folder)
2. Activate the virtual environment
	> venv\Scripts\activate
3. To run the different files using jupyter notebook, 
	> jupyter notebook
4. To stop the virtual environment
	> deactivate
